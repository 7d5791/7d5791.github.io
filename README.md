# Visualización de Datos

Sitio académico personal para presentar tareas y reflexiones del curso de Visualización de Datos.

## Estructura

- `index.html`: página principal
- `assets/css/styles.css`: estilos del sitio
- `assets/js/main.js`: interacciones básicas
- `assets/img/`: recursos visuales

## Cómo usar

1. Descarga o clona el proyecto.
2. Abre `index.html` en tu navegador.
3. Personaliza textos, enlaces e imágenes según tu necesidad.
